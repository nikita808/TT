Перед началом использования программы необходимо зарегистрироваться на сайте https://dadata.ru и в личном кабинете
найти API ключ, и при первом запуске программы ввести его в командную строку. (В дальнейшем его можно будет изменить
в настройках).

Инструкция по запуску:
    1. Создать python virtual environment
    2. Активировать venv
    3. Установить зависимости (pip install -r r.txt)
    4. Запустить программу (python main.py)
    5. Пользоваться
